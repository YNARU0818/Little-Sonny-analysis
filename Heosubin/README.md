# ⚽ Project: Little-Sonny (차세대 유망주 예측 서비스)

본 프로젝트는 FIFA 축구 선수 데이터를 분석하여, 특정 선수가 향후 대성할 가능성이 있는 **'유망주(Prospect)'**인지 여부를 예측하는 머신러닝 모델 개발 프로젝트입니다.

## 📋 프로젝트 개요
- **목적**: 선수들의 신체 조건, 기술 스탯, 심리적 요인 등을 종합하여 유망주(1)와 일반 선수(0)를 분류
- **데이터셋**: FIFA 선수 정보 기반 (나이, 포지션, 6대 핵심 스탯 및 세부 기술 스탯 등)
- **평가 지표**: Macro F1 Score (불균형 데이터셋 고려)

## 🔍 주요 분석 관점 (EDA)
데이터 분석 과정에서 다음과 같은 핵심 가설과 관점을 바탕으로 탐색적 데이터 분석(EDA)을 수행했습니다.

1. **나이(Age)와 유망주 상관관계**:
   - 16~18세 구간에서 유망주 비율이 가장 높으며, 나이가 들수록 유망주 선정 비율이 급격히 하락하는 경향을 확인하여 연령별 가중치 필요성을 파악했습니다.
2. **핵심 스탯 그룹화 및 파생 변수**:
   - **Scouting Score**: 단순 스탯 나열이 아닌 공격 결정력, 속도, 기술 등을 조합한 자체 지표를 생성하여 모델의 변별력을 높였습니다.
3. **포지션 그룹화 (`Pos_Group`)**:
   - 15개 이상의 세부 포지션을 Forward, Midfielder, Defender, GK로 단순화하여 학습 효율을 최적화했습니다.

## 🛠️ 모델링 전략 (Modeling Strategy)

본 프로젝트는 자동화 프레임워크(AutoGluon)를 통한 베이스라인 탐색 후, 예측 제어력을 높이기 위해 **수동 가중치 앙상블(Weighted Ensemble)** 방식을 최종 채택했습니다.

### 1. AdaBoost 중심 가중치 앙상블
- **전략**: 오분류된 샘플에 집중하는 AdaBoost의 특성을 활용하고, 모델별 강점을 조합하여 Soft Voting 수행
- **가중치 배분**:
  - **AdaBoost (0.4)**: 핵심 유망주 탐지 및 오차 보정 강화
  - **RandomForest (0.3)**: 모델의 안정성 확보 및 과적합 방지
  - **LightGBM (0.3)**: 대량 데이터의 비선형적 특성 학습

### 2. 보조 모델링 (LightAutoML)
- `Fast` 프리셋을 활용하여 피처 중요도(Feature Importance)를 빠르게 파악하고, 주요 변수(Pace, Finishing, Potential 등)가 모델에 미치는 영향을 시각화했습니다.

### 3. 임계값(Threshold) 최적화
- 단순 0.5 기준이 아닌, 검증 데이터에서 Macro F1 점수를 극대화할 수 있는 **최적의 임계값(예: 0.3434)**을 찾아 최종 예측에 적용했습니다.


## 📊 모델별 성능 및 특징 비교

각 노트북 파일에서 시도한 모델들의 성능과 주요 차이점은 다음과 같습니다.

| 모델 구분 | 사용 기술/프레임워크 | 주요 특징 | 성과 (Macro F1) | 비고 |
| :--- | :--- | :--- | :--- | :--- |
| **Manual Ensemble** | **AdaBoost + RF + LGBM** | 모델별 가중치 부여 (4:3:3) 및 임계값 최적화 | **0.76~0.77 (추정)** | **최종 채택 전략** (제어력 높음) |
| **LightAutoML** | **LAMA (Fast Preset)** | 피처 중요도 분석 및 빠른 베이스라인 구축 | **0.7683** | 변수 영향력 파악에 최적 |
| **AutoGluon** | **Stacking Ensemble** | 다계층 앙상블을 통한 자동 최적화 | **0.75~0.76** | 성능은 높으나 모델 제어 어려움 |

### 🔍 모델별 상세 차이점
1.  **가중치 앙상블 (ADA 노트북)**: 획일화된 자동화 모델링에서 벗어나, 유망주 분류에 강점을 보이는 **AdaBoost에 높은 가중치(0.4)**를 부여했습니다. 오분류된 데이터에 집중하는 AdaBoost의 특성을 활용해 성능을 극대화했습니다.
2.  **LightAutoML**: 데이터의 특성을 가장 잘 반영하는 상위 20개 변수를 추출하는 데 유용했으며, 특히 **나이(Age)와 잠재력(Potential)**의 상관관계를 가장 정교하게 잡아냈습니다.
3.  **AutoGluon**: 여러 모델을 쌓아서 학습하는 'Stacking' 방식이지만, 본 프로젝트에서는 보다 정밀한 튜닝을 위해 최종 단계에서는 주석 처리하고 수동 앙상블로 전환하였습니다.

## 📈 분석 결과 및 성능

- **최종 모델링**: AdaBoost(0.4) + RF(0.3) + LGBM(0.3) 앙상블을 통해 불균형 데이터셋에서도 안정적인 Macro F1 Score 달성
