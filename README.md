## ⚽ Project 1: Soccer Prospect Prediction

FIFA 선수 데이터를 활용하여 향후 대성할 가능성이 있는 **'유망주(Prospect)'** 여부를 예측하고 랭킹을 도출하는 프로젝트입니다.

### 🔍 탐색적 데이터 분석 (EDA)

* **나이(Age)의 상관관계**: 16~18세 구간에서 유망주 비율이 압도적이며, 나이가 들수록 확률이 급격히 하락하는 음의 상관관계를 확인.
* **Scouting Score 설계**: 단순 스탯 나열이 아닌 공격 결정력, 속도, 기술 등을 조합한 자체 파생 변수를 생성하여 모델의 변별력을 확보.
* **포지션 그룹화(PosGroup)**: 15개 이상의 세부 포지션을 Forward, Midfielder, Defender, GK로 단순화하여 학습 효율 최적화.

### 🛠 모델링 및 성능 (Stratified 5-Fold CV)

여러 전통 머신러닝 모델을 비교 분석한 결과, **AdaBoost**가 유망주 누락(False Negative) 방지에 가장 우수한 성능을 보였습니다.

| Model | Accuracy | F1-score | ROC-AUC | Precision | Recall |
| --- | --- | --- | --- | --- | --- |
| **AdaBoost** | **0.7854** | **0.6997** | 0.8451 | 0.7077 | **0.6927** |
| LogisticRegression | 0.7807 | 0.6745 | **0.8497** | 0.7276 | 0.6294 |
| RandomForest | 0.7748 | 0.6667 | 0.8387 | 0.7162 | 0.6248 |

### 🤖 AutoGluon & 최종 전략

* **AutoGluon 앙상블**: `WeightedEnsemble_L2` 모델이 Validation F1 0.7273을 기록하며 안정적인 성능 확인.
* **Feature Importance**: **Age(0.322)**가 압도적이며, Stamina(0.0166), Height(0.0135), Agility(0.0083) 순으로 영향력을 미침.
* **최종 모델**: **AdaBoost(0.4) + RF(0.3) + LGBM(0.3)** 가중치 앙상블에 최적 임계값(**Threshold 0.3434**)을 적용하여 최종 예측 수행.

## ✍️ Project 2: Student Performance Analysis

합성 데이터의 논리적 결함을 역추적하고, 학생 성적에 영향을 미치는 실제적 요인을 분석했습니다.

### 🔍 데이터 품질 및 인사이트

* **논리적 모순 발견**: 인터넷 미연결(`Internet Connection: No`) 상태이나 온라인 강의(`Online Video`)를 수강하는 모순 데이터 8,730건 식별.
* **데이터 편향**: 전공별 성별 비중이 인위적으로 균등(각 33.3%)하게 분포된 합성 데이터의 특성 파악.
* **핵심 요인**: 절대적 수면 시간보다 **수면의 질**이 성적과 높은 상관관계를 보였으며, 공부 시간이 짧아도 고득점인 그룹은 **코칭(Coaching)** 비율이 압도적으로 높음.

### 🛠 전처리 전략

* **Clipping 처리**: 20점 이하 및 100점 이상 구간의 이상치를 삭제하지 않고 임계값으로 고정하여 데이터 분포 보존.

## 🚗 Project 3: Used Car Price Prediction

18만 개의 대규모 데이터를 분석하여 중고차 시세를 예측하는 회귀 프로젝트입니다. (MAE 22% 감소, R2 205% 향상)

### 🛠 정교한 피처 엔지니어링 (Deep Dive)

* **Regex 엔진 정보 추출**: 비정형 텍스트인 `engine` 컬럼에서 **Horsepower(HP)**, **Displacement(L)**, **Cylinders** 정보를 정규표현식으로 정밀 파싱.
* **Mileage Intensity**: `milage / (2024 - model_year)` 파생 변수를 통해 연식 대비 차량 혹사 정도를 수치화.
* **Target Encoding**: 브랜드/모델 변수는 이상치 영향을 방지하기 위해 **중위값(Median)** 기준으로 매핑.

### 🚫 도메인 기반 이상치 제거 (Outlier Removal)

* **상한선**: $1,000,000 이상 초고가 및 일반 브랜드 $200,000 초과 매물 제거.
* **하한선**: $4,000 미만 폐차 수준 및 럭셔리 브랜드 $30,000 미만(사기 의심) 매물 제거.
* **모순 데이터**: 2010년 이전 연식인데 주행거리 100마일 미만인 '좀비 카' 필터링.

### 📈 모델링 및 성과

* **전략**: **CatBoost**(범주형 데이터) + **LightGBM**(수치형 데이터) + **XGBoost** 앙상블 및 5-Fold 교차 검증.
* **로그 변환**: 가격(Target) 데이터의 편향을 해결하기 위해 `log1p` 적용.
* **결과**: 로그 스케일 기준 검증 데이터셋 **R2 점수 0.8X 이상** 달성.

## 🎯 최종 결론 (Conclusion)

1. **데이터 품질의 중요성**: 단순 모델 튜닝보다 도메인 지식에 기반한 정교한 데이터 클리닝과 논리 검증이 모델 성능을 결정짓는 핵심 요인임을 확인했습니다.
2. **합성 데이터의 한계**: 합성 데이터는 샘플을 보강해주지만 원본의 노이즈를 복제하는 부작용이 있으므로, 분석가는 수치 너머의 현실적 개연성을 판단해야 합니다.
3. **데이터 정제의 재정의**: 정제는 단순 삭제가 아니라 비상식적인 값을 걸러내고 데이터의 본질적 가치를 회복하는 과정임을 학습했습니다.
