## 1️⃣ [Classification] Soccer Prospect Prediction

> **"나이라는 제약 조건을 넘어 최적의 유망주 랭킹을 도출하다"**

### 🔍 핵심 전략

* **데이터 설계**: 단순 스탯을 넘어 공격력, 속도, 기술을 결합한 **Scouting Score** 파생 변수 생성.
* **모델링 (AutoGluon)**: 개별 모델의 한계를 극복하기 위해 **Stacking** 및 **Weighted Ensemble**을 적용하여 모델의 견고함 확보.
* **최적화**: 유망주 누락을 방지하기 위해 Macro F1을 극대화하는 **최적 임계값(0.3434)** 설정.

### 📈 주요 성과

* **Feature Importance**: **나이(Age, 0.322)**의 지배적 영향력을 확인하고, Stamina와 Height 등 신체 지표의 보조적 역할을 증명.
* **결과**: 고도화된 모델링을 통해 실질적인 유망주 판별의 정밀도와 재현율 동시 확보.

## 2️⃣ [Insight] Student Performance Analysis

> **"합성 데이터 속 숨겨진 모순을 찾아 분석의 당위성을 입증하다"**

### 🔍 핵심 전략

* **무결성 검증**: 인터넷 미연결 상태에서 온라인 강의를 수강하는 등의 **논리적 모순 데이터(8,730건) 역추적**.
* **데이터 정제**: 20점 이하/100점 이상 데이터를 삭제하는 대신 **Clipping 기법**을 적용하여 합성 데이터 특유의 분포 왜곡 방지.
* **학습 효율 분석**: 공부 시간의 절대량보다 **수면의 질**과 **전문 코칭**이 성적에 미치는 영향 분석.

### 📈 주요 성과

* **인사이트**: '공부 시간'보다 '생활 습관'과 '외부 환경'이 성적의 결정적 요인임을 밝혀내 분석의 깊이 확장.
* **결과**: 신뢰할 수 없는 노이즈를 제거하여 실제 성적 예측 모델의 데이터 기반 마련.

## 3️⃣ [Regression] Used Car Price Prediction

> **"비정형 노이즈를 정제하여 초기 모델 대비 성능을 205% 향상시키다"**

### 🔍 핵심 전략

* **Regex 엔진 파싱**: 비정형 `engine` 컬럼에서 **마력(HP)**, **배기량(L)** 등의 핵심 수치 정보를 정규표현식으로 정밀 추출.
* **Domain Filtering**: 브랜드 가치에 맞지 않는 허위 고가/저가 매물 및 2010년 이전 연식의 '좀비 카' 데이터 제거.
* **Target Encoding**: 브랜드/모델 변수를 **중위값(Median)** 기준으로 매핑하여 이상치에 대한 저항성 확보.

### 📈 주요 성과

* **성능 비약적 향상**: 초기 R2 Score 0.26 수준에서 전처리 및 앙상블 후 **최종 0.79~0.80 수준**으로 성능 개선.
* **결과**: 로그 변환(`np.log1p`)과 CatBoost/LightGBM 앙상블을 통해 **초기 대비 205%의 성능 향상** 달성.
